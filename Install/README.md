# Installation

## Prerequisites

- [Git](https://git-scm.com/)
- [Claude Code CLI](https://claude.ai/code)

## Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/<your-username>/ClaudeScaffold.git
   cd ClaudeScaffold
   ```

2. Review the project structure and customize files under `scaffold/` to fit your needs.

3. Follow any additional setup steps described in the root `README.md`.

## Troubleshooting

If you run into issues, check the following:

- Ensure Git is installed and available on your PATH.
- Verify you have the correct permissions for the target directory.
